1.서버
   - port :7500
   - serverStart()사용자정의 메서드 : 소켓생성,접속대기
2.클라이언트
   - port : 7500
   - Thread생성
   - 접속시도
---------------------- 테 스 트  1 --------------------------
3. 클라이언트
   - 닉네임(nickname)을 서버에 전송
4. 서버
   - 병행처리를 하기위한 client 객체생성(사용자정의 클래스)
   - Vector에 생성된 client 객체 추가 (객체카운트)
   - 각각의 client에 대한 thread객체생성
   - 쓰레드 시작
   - run()메서드에서  nickname을 읽어, txt_list에 출력
--------------------- 테 스 트 2 ---------------------------
   - 각각의 client에게 접속된 사용자(nickname)을 보냄 :broadcast
5. 클라이언트
   - 서버로부터 받은 nickname을 출력화면에 보여줌
--------------------- 테 스 트 3 ---------------------------
   - txt_input 또는 btn_send를 이용해서 대화전송
6. 서버
   - 대화내용을  read해서 txt_list에 출력
   - 모든 사용자에게 대화내용을 전송 
--------------------- 테 스 트 4 ---------------------------
   - 퇴장시 퇴장메세지를 알림(동기화처리)
